# FocusNest Frontend API Setup

Welcome to the FocusNest Frontend interface. 
It is engineered dynamically using React, TailwindCSS, Framer Motion, and shadcn/ui.

## Local Development
Make sure you are running the backend in the `server` folder (`npm run dev`) simultaneously on port 3000. 
Vite proxy configuration handles routing local frontend fetch requests directly to the API endpoints.

```bash
# Start Vite development server
npm run dev
```

Build for production using:
```bash
npm run build
```
